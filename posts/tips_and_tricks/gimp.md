# GIMP

## Resynthesizer

```sh
pamac install babl-git gegl-git python2-gimp gimp-plugin-resynthesizer
```

Adds plugin to GIMP with tools like:
- Heal selection tool (removes selected parts from image)
- Enlarge image and synthesize to sharpen
